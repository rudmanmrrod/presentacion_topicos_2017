¿Cómo se instala el sistema de modelado de tópicos? Parte 1

  • Lo primero es descargar/clonar el repo, la última versión disponible en github: https://github.com/rudmanmrrod/modelado_topicos
  • Instalar los requierimientos con python: pip install -r requirements.txt
  • Meternos en la carpeta vsm y ejecutar el comando python setup.py develop
  • Instalar las stopwords del nltk, o nos metemos en python y usamos import nltk y luego nltk.download() para usar la interfaz gráfica o de una vez usamos el comando python -m nltk.downloader stopwords
  • En la siguiente parte instalaremos el Freeling para hacer el pre-procesamiento
  • No nos debemos preocupar por instalar el LDA, ya esta compilado y metido en el proyecto, pero en caso que por arte de magia no funcione más adelante se explicará como instalarlo
