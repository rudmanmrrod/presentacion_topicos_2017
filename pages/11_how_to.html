¿Cómo se usa el sistema de modelado de tópicos?

El sistema consta de 3 módulos importantes, 4 si tomamos en cuenta el registro de usuario. Estos 3 módulos son:

  • Carga de Corpus
  • Perfiles de Procesamiento y
  • Visualización

Carga de Corpus: Consiste en un módulo con dos (2) submódulos, Carga y Ver. En el primero se crea un directorio y se adjuntan n archivos y en el segundo se puede ver la lista de directorios con corpús creados, donde se pueden añadir más archivos o eliminar el registro con los archivos.

Perfiles de Procesamiento: Es un módulo con tres (3) funcionalidades principales, crear perfiles de procesamiento, listar los perfiles de procesamiento y ejecutar el pre-procesamiento vía interfaz web. Dentro del apartado de listar, se puede editar el perfil, eliminar o ver los resultados.

Visualización: Es el apartado que permite ver el resultado de la ejución del LDA, mostrando primero la lista de tópicos y luego una visualización con barras de la distribución de los tópicos, así como también ver la distribución de las palabras de uno, varios o todos los tópicos para un documento en particular. A esta sección se accede mediante el botón ver resultados en las lista de perfiles, cabe destacar que este botón se muestra luego de que se ejecute al menos una vez el pre-procesamiento.
